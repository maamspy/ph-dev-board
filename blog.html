Q1: What are the different ways to select an element in the DOM?

In JS, there are multiple ways to select elements in the DOM. Let me give you some example:

1. getElementById()
This selects an element by its unique id.
2. getElementsByClassName()
This selects all elements with the specified class name and returns a live HTMLCollection.
3. getElementsByTagName()
This selects all elements of a given tag name (like div, section, p etc.) and returns a live HTMLCollection.
4. querySelector()
This selects the first element that matches the CSS selector and returns a single element.
5. querySelectorAll()
That selects all elements that match the CSS selector and returns a static NodeList.

Q2: What is the difference between innerHTML, innerText, and textContent?

innerHTML:
innerHTML gives all HTML content inside an element including all HTML tags.
It can modify the HTML structure inside an element.

innerText:
innerText gives the visible text of an element only.
It can modify the text, not the HTML structure.

textContent:
textContent gives the text contents of an element while ignoring any HTML tags.
It never considers CSS styles like hidden content and gives the entire text inside the element where innerText can't give hidden elements or text which is not displayed.

Q3: What is event delegation in the DOM?

Event delegation is a way of handling events in the DOM by attaching a single event listener to a parent element instead of adding listeners to every child element. When an event happens on a child element, it bubbles up to the parent and the parent can handle it.

Q4: What is event bubbling in the DOM?

Event bubbling is a process in the DOM where an event starts from the target element and then goes up to its parent nodes, then to its grandparent nodes and so on till the way up to the root of the document.

Q5: How do you create, add, and remove elements using JavaScript?

To create an element, we use document.createElement() while passing the tag name of the element we want to create.

We can add text or HTML to the created element using textContent or innerHTML. Once the element is created, we need to add it to the DOM using appendChild().

To remove an element, we use removeChild() on the parent element.
